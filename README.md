# [RSLang](https://alexiykami.github.io/rslang/)

RS Lang is an application for learning foreign words, which includes an electronic textbook with a database of words to learn, mini-games for repeating them, a statistics page for tracking individual progress.

### 🛠 Tech stack: 
- [Webpack](https://webpack.js.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Eslint](https://eslint.org/)
- [Prettier](https://prettier.io/)
- [Sass](https://sass-scss.ru/documentation/)

### 📚 Library used:
- [Chart.js](https://www.chartjs.org/)
- [Lodash](https://lodash.com/)
